# Tic-Tac-Toe
This is a simple Tic-Tac-Toe game implemented using React, showcasing fundamental concepts of React components, state management, and event handling. The game follows the classic rules of Tic-Tac-Toe, where two players take turns to mark 'X' or 'O' in a 3x3 grid, aiming to achieve three marks in a row horizontally, vertically, or diagonally.

Features
Responsive Design: The game is designed to be responsive and can be played seamlessly on various screen sizes.
Interactive Gameplay: Players can click on the grid cells to make their move, and the game updates the board accordingly.
Winning Detection: The game automatically detects a winner or a draw and displays the result.
Turn Indicator: The current player's turn is visually indicated to enhance the user experience.
Technologies Used
React: The game is built using the React library, enabling the creation of reusable components and efficient state management.
CSS: Styling is done using CSS to provide an appealing and user-friendly interface.
